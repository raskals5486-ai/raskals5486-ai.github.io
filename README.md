# Taxi Demand Radar SPB landing page

Статический HTTPS-сайт для публичного описания проекта и заявки на API Яндекс Go.

## Локальный просмотр

```powershell
python -m http.server 8080 --directory site
```

Открыть `http://localhost:8080`.
