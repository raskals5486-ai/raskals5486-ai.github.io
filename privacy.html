2. Назначение продукта

Taxi Demand Radar SPB — независимый информационно-аналитический сервис для оценки потенциальной транспортной нагрузки в Санкт-Петербурге. Прогноз не является гарантией заказа, заработка или повышающего коэффициента.

3. Данные сайта

Текущая статическая версия сайта не содержит форм регистрации, оплаты или отправки пользовательских маршрутов. Сайт самостоятельно не устанавливает рекламные или аналитические cookie. Технические журналы доступа могут обрабатываться выбранным хостинг-провайдером для обеспечения безопасности и доступности сайта.

4. Данные внешних API

Проект планирует использовать только официально разрешённые API и договорные источники. API Яндекс Go может применяться после получения ключа и согласования условий — для отображения конечным пользователям ориентировочной стоимости, времени подачи и параметров поездки. Частота запросов, срок хранения и производные расчёты ограничиваются условиями правообладателя и выданного доступа.

5. Какие данные не требуются

  • ФИО, телефон, email или платёжные данные пассажиров;
  • история заказов и поездок конкретного пассажира или водителя;
  • учётные данные, токены или сведения из приложения Яндекс Про;
  • данные, полученные перехватом трафика, OCR или автоматизацией сторонних приложений.

После запуска приложения его фактические разрешения и состав телеметрии будут раскрыты на этой странице. Секретные ключи API не размещаются в APK и хранятся на серверной стороне.
